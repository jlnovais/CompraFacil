CompraFacil Webservice (asmx) access demo

Prerequisites:

- Code editor (Visual Studio 2017 recommended)
- c# 7.1

To use the CompraFacil Webservice you can add a "Connected Service" to the project.

Update the appsettings.json file with the correct access details.
